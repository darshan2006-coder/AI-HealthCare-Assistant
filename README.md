# 🩺 HealthBot AI

> An AI-powered healthcare assistant that performs intelligent symptom assessment, evaluates patient risk, generates Electronic Health Records (EHR), and provides a secure, responsive healthcare experience.

# 🌐 Live Demo

🚀 Click here to explore the website : https://ai-healthcare-assistant-kkwh.onrender.com

## 📌 Overview

HealthBot AI is a full-stack healthcare web application designed to assist users with preliminary symptom assessment through an AI-driven workflow. The application securely manages patient sessions, evaluates health risks, maintains consultation history, and generates downloadable EHR reports in PDF format.

Built with a focus on usability, security, and responsive design, the project demonstrates modern full-stack development practices and clean UI implementation.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/darshan2006-coder/AI-HealthCare-Assistant.git
cd AI-HealthCare-Assistant
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file in the project root.

```env
PORT=3000
SESSION_SECRET=your_session_secret
GEMINI_API_KEY=your_api_key
```

### Run the application

```bash
npm start
```

Visit:

```text
http://localhost:3000
```
